# Быстрый старт

## Шаг 1: Установите зависимости

```bash
cd frontend
npm install
```

## Шаг 2: Убедитесь, что бэкенд запущен

Бэкенд должен быть запущен на http://localhost:8000

```bash
# В корневой директории проекта
uvicorn src.app:app --reload --host 0.0.0.0 --port 8000
```

## Шаг 3: Запустите фронтенд

```bash
npm run dev
```

## Шаг 4: Откройте браузер

Перейдите на http://localhost:3000

## Первые шаги

1. **Зарегистрируйтесь** - создайте новый аккаунт
2. **Войдите** - используйте свои учетные данные
3. **Создайте папку** - нажмите "+ New Folder"
4. **Загрузите скрипт** - нажмите "+ New Script" и выберите .py файл
5. **Просмотрите скрипт** - кликните на скрипт для просмотра содержимого

## Решение проблем

### Ошибка подключения к API

- Убедитесь, что бэкенд запущен на порту 8000
- Проверьте консоль браузера на наличие ошибок CORS

### Ошибки при установке

- Убедитесь, что Node.js >= 18.0.0 установлен
- Попробуйте удалить `node_modules` и переустановить:
  ```bash
  rm -rf node_modules package-lock.json
  npm install
  ```
